# 竞争力分析服务 (Competitiveness Analysis Service)

## 概述

本项目是 AI 驱动的汽车行业情报平台的一个子服务，负责对采集到的文章进行深度分析，提取结构化的竞争力情报。

服务利用大型语言模型（LLM）分析文章内容，识别其中涉及的汽车品牌、技术创新点、市场趋势等，并将结果结构化地存入数据库。

## 主要功能

- **自动分析**: 接收文章ID，自动从主数据库获取文章内容并启动分析流程。
- **结构化提取**: 从非结构化的文本中提取有关品牌、技术、市场等维度的信息。
- **后台处理**: 分析任务在后台异步执行，不阻塞API请求。
- **API接口**: 提供标准的RESTful API，用于触发分析、查询结果。
- **独立配置**: 服务拥有独立的配置文件（`.env`），用于管理LLM API密钥等敏感信息。

## 技术栈

- **框架**: FastAPI
- **数据模型**: Pydantic, SQLAlchemy
- **配置管理**: pydantic-settings
- **AI模型**: 支持多种LLM（当前默认使用智谱AI）

## 如何使用

通过API端点与服务交互。服务启动后，API文档将位于 `/docs`。
